How to connect heroku with github with automatic deployment

Github: Create a repo like normal, add/commit/push as usual

Command: heroku create -> create the project in heroku, it will give you a .git link

Heroku: Deploy tab - Deployment method - Choose github

Heroku: connect it to the github repo, and choose the branch to deploy with

Heroku: Automatic deploys - Enable automatic deploys

Ongoing

git add/commit/push project to development branch

Github: create a PR to merge development branch to production branch

Changes will automatically updated in Heroku
